4. CIA Triad
Confidentiality: Prevent unauthorized access.

Integrity: Ensure accuracy and trustworthiness.

Availability: Keep systems accessible.

5. NIST Cybersecurity Framework (CSF)
Voluntary, risk-based, globally adopted.

Five Core Functions: Identify, Protect, Detect, Respond, Recover.

Importance: Provides a comprehensive, flexible approach to risk management.

6. OWASP Security Principles
Focus: Web application security.

Community-driven resources: Guides, tools, methodologies.

OWASP Top 10: SQL injection, broken authentication, XSS, insecure deserialization.

Importance: Helps developers prevent common web vulnerabilities.

7. Security Audits
Definition: Independent examinations of security controls and processes.

Key Elements: Scope & goals, risk assessment, controls assessment, compliance, communication of results.

Examples: GDPR, PCI DSS.

Importance: Identifies weaknesses, ensures compliance, and drives continuous improvement.
